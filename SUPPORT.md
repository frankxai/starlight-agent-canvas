# Support

Use the channel that matches the problem.

## Setup Or MCP Help

Open a Setup / MCP help issue and include:

- operating system
- Node and pnpm versions
- `pnpm doctor` output
- `pnpm doctor:json` summary
- whether Codex, Claude Desktop, Gemini, or another MCP host is involved
- redacted config snippets only

## Bugs

Open a bug report with:

- exact steps to reproduce
- expected behavior
- observed behavior
- affected input type, node kind, action, or MCP tool
- verification commands already run

## Feature Or Integration Requests

Open a feature or integration request and explain:

- the human workflow
- the agent workflow
- source types involved
- safety boundary
- expected export or MCP behavior

## Security

Do not open a public issue for vulnerabilities, secrets, or exploit details. Follow `SECURITY.md`.
