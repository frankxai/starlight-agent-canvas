# Code Of Conduct

Starlight Agent Canvas is an OSS-first local tool for human and agent workflows. Keep the project useful, safe, and welcoming.

## Expected Behavior

- Be respectful and concrete.
- Assume good intent, but verify claims with evidence.
- Keep critique focused on the code, docs, design, security boundary, or user workflow.
- Respect privacy. Do not post private transcripts, tokens, canvas exports, logs, or user data.
- Prefer small, reviewable changes with clear verification.

## Unacceptable Behavior

- Harassment, abuse, threats, or discriminatory language.
- Publishing secrets, private data, exploit details, or private canvas contents.
- Pushing destructive tools, external-posting flows, credential mutation, scraping, or payment/account actions into v0.1 without explicit product review.
- Repeatedly derailing issues or PRs away from the project scope.

## Enforcement

Maintainers may edit, hide, close, or lock issues and pull requests that violate this code. Serious or repeated violations may result in a ban from project spaces.

Security-sensitive reports should follow `SECURITY.md`, not public issue threads.
